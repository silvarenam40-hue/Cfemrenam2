Dashboard CFEM

Este repositório contém o código do dashboard desenvolvido com Streamlit para análise de Metadados CFEM.

Como executar localmente

  1. Clone o repositório:
    2. git clone https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git
  2. Instale as dependências necessárias (recomenda-se usar um ambiente virtual):
    3. pip install -r requirements.txt
  3. Execute o dashboard:
    4. streamlit run dashboard_cfem.py

O aplicativo estará disponível no navegador, geralmente em http://localhost:8501.

Sobre

Este projeto utiliza Python e Streamlit para visualização de dados relacionados à CFEM.
